== Description ==

Send notification buddypress for all member when publish woocommerce product.
You can send notification to your friends or all users in the dashboard woocommerce product.

== Installation ==

1. Upload `Bp-notification-woocommerce` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Choose send notification for: All user or My friends in the dashboard woocommerce product
